# Adaptive Query-Aware Chunking and Explainable Retrieval Framework

## Overview

A domain-independent retrieval framework that dynamically selects retrieval granularity based on query semantics.

## Features

- Adaptive Chunking
- Semantic Retrieval
- FAISS Vector Search
- Explainable Retrieval
- Query-Aware Granularity Selection

## Technologies

- Python
- FAISS
- SentenceTransformers
- NLTK
- PyPDF2

## Run

pip install -r requirements.txt

python main.py
